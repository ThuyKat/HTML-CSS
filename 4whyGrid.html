Option1: Using float

Header: place on top and do nothing with it because its staying on top according to normal stacking order. Next, we have two sections: right and left sections. What we are going to do is that placing a "wrapper" around the left section, float it to the left and then two sections inside it Then we are going to float the "Main" content to the right. And of course because we did that, we take all these elements out of their document flow, it will collapse that makes the footer sinked behind the content. So what we need to do is that we need to clear floats, place that at the bottom on top of the footer to resume the normal document flow.

Option 3: Using Grid

There is only one div to group all items call grid grabper. We dont have to nest any elements inside other elements. The cool thing is the order of element in HTML is not important. So we have the flexibility. We can re-arange elements depends on the screen size. CSS grid treats the webpage like girds.
